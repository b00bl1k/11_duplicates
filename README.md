# Anti-Duplicator

Script to search for duplicate files. Search criteria is same name and size.

# Example of usage:

```bash
$ python duplicates.py .
Duplicate file 'master' with size 41:
 ./.git/refs/heads/master
 ./.git/refs/remotes/origin/master
```
